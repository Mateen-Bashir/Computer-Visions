**Computer Vision Lab: Image Enhancement Techniques**

**Overview**



This project demonstrates several image enhancement techniques used in Computer Vision using Python, OpenCV, NumPy, and Matplotlib. These techniques are used to improve image quality, highlight important features, and prepare images for further analysis in fields such as medical imaging, agriculture, surveillance, and autonomous systems.



**Technologies Used**



Python



OpenCV



NumPy



Matplotlib



**Image Processing Techniques Implemented**

1\. Image Negative



Concept



Image negative transformation inverts pixel intensities.



**Purpose**



Highlight bright objects in dark regions



Improve visibility in medical images.



**Applications**



X-ray analysis



Night surveillance enhancement



2\. Log Transformation



**Concept**



Log transformation expands dark pixel values and compresses bright pixels.



**Purpose**



Enhance low-intensity regions.



**Applications**



Astronomical image analysis



Satellite imagery



Dark scene enhancement



3\. Gamma Correction



**Concept**



Gamma correction adjusts image brightness nonlinearly.



**Applications**



Photography brightness correction



Medical imaging



Thermal image enhancement



4\. Contrast Stretching



**Concept**



Contrast stretching expands the dynamic range of image intensities.



**Applications**



License plate recognition



Foggy road image enhancement



Self-driving car perception



5\. CLAHE (Contrast Limited Adaptive Histogram Equalization)



**Concept**



CLAHE improves contrast locally rather than globally.



**Applications**



Document restoration



OCR preprocessing



Medical imaging
