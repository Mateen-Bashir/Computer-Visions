## 🔍 Overview

This repository implements a progressive learning path from basic Python/OpenCV operations to advanced pattern recognition and neural networks. Each lab follows a structured approach: Journey (theory) → Apply (implementation) → Verify (case studies) → Assess (evaluation).

## 🛠️ Techniques Covered

### **1. Image Processing Fundamentals**
- Image I/O operations (read, display, save)
- Color space conversions (BGR, RGB, HSV, Grayscale)
- Geometric transformations (resizing, cropping)
- Point processing techniques

### **2. Scene Enhancement Techniques**
- **Contrast Enhancement**
  - Contrast stretching (min-max normalization)
  - Histogram equalization
  - Adaptive histogram equalization (CLAHE)
  - Gamma correction
- **Intensity Transformations**
  - Image negative transformation
  - Log transformation
  - Power-law transformations

### **3. Feature Extraction Methods**
- **Edge Detection**
  - Sobel operator (horizontal/vertical gradients)
  - Prewitt operator
  - Canny edge detection
- **Texture Analysis**
  - Gabor filters (multi-orientation, multi-scale)
  - Local Binary Patterns (LBP)
  - Autocorrelation analysis

### **4. Object Detection & Recognition**
- **Traditional Methods**
  - Haar Cascade classifiers (face detection)
  - Template matching
  - HOG features
- **Deep Learning Methods**
  - YOLOv8 (real-time object detection)
  - MobileNet SSD
  - Pre-trained model implementation

### **5. Pattern Recognition Techniques**
- **Data Preprocessing**
  - Auto-scaling (Z-score normalization)
  - Feature standardization
  - One-hot encoding
- **Dimensionality Reduction**
  - Principal Component Analysis (PCA)
- **Clustering**
  - K-Means clustering
- **Feature Weighting**
  - Statistical feature importance
  - Model-based feature selection

### **6. Neural Network Implementations**
- **Architectures**
  - Artificial Neural Networks (ANN)
  - Convolutional Neural Networks (CNN)
  - Multi-layer perceptrons
- **Applications**
  - Handwritten digit recognition (MNIST)
  - Alphabetical character recognition (EMNIST)
  - Face recognition (LBPH)

### **7. Advanced Topics**
- **Content-Based Image Retrieval (CBIR)**
  - Color histogram-based retrieval
  - Texture-based retrieval
  - Similarity metrics (Euclidean, Cosine, Chi-square)
- **Optimization Algorithms**
  - Particle Swarm Optimization (PSO) for object localization
- **Biometric Systems**
  - Fingerprint enhancement
  - Face detection and recognition
