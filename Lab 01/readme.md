**Real-Time QR Code Scanner**

Description

Scans QR codes using OpenCV and logs entry data.



**Features**

QR detection

Decode QR content

Entry logging

Duplicate prevention

Timestamp storage

Real-time webcam display

Requirements

pip install opencv-python



**Run**

python app.py



**Output**

Webcam window with QR detection

qr\_log.txt containing scanned data
